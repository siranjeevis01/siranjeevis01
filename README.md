## 🚀 About Me

I'm a **Backend Software Developer** with hands-on experience designing, developing, and maintaining RESTful APIs using **ASP.NET Core (.NET 8)** and **Entity Framework Core**. I'm passionate about building clean, maintainable systems using **SOLID Principles**, **Repository Pattern**, and **Layered Architecture**.

**What I bring to the table:**
- 🔧 **5+ production-grade backend projects** with real-world architecture
- ⚡ **40% latency reduction** using RabbitMQ async messaging
- 📦 **Containerized deployments** with Docker & Docker Compose
- 🔐 **Secure API design** with JWT authentication & role-based authorization
