I'm a **Software Developer at Neolysi Technologies, Chennai**, working on **RESTful APIs** with **ASP.NET Core (.NET 8)** and **Entity Framework Core** for enterprise business applications. I care about clean, maintainable backend systems built on **SOLID Principles**, **Repository Pattern**, and **Layered Architecture**.
